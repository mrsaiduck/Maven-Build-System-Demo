Build systems automate the process of compiling source code into executable programs.
A build system manages dependencies, runs tests, and packages the final artifact.
Maven is a popular build system for Java projects.
Maven uses a declarative approach where you describe what you want, not how to build it.
The build lifecycle in Maven consists of phases like compile, test, package, and install.
Each phase is made up of plugin goals that do the actual work.
Dependencies are downloaded automatically from remote repositories like Maven Central.
